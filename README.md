# Mission
Repository of an example using React, NodeJS, MongoDB and so on.

## Setup

`git clone https://github.com/AugustoCalaca/mission.git`

`cd mission`

### Install

```
yarn
```

## Server
```
yarn server:start
```

#### Seed data into database
```
yarn seed
```

## Web
```
yarn web:start
```

In your browser access localhost:6060
