# Mission
Repository of an example using React, NodeJS, MongoDB and so on.

## Install

```
yarn
```

## Server
```
yarn server:start
```

#### Seed data into database
```
yarn seed
```

## Web

### Stack

* React/Hooks
* Typescript
* React Router
* Material UI

### Run

```bash
yarn web:start
```

In your browser access localhost:3030
