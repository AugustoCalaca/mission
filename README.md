# Mission
Repository of an example using React, NodeJS, MongoDB and so on.
It is based on courses that have the following structure:

```
title:       string
subtitle:    string
startedAt:   datetime
description: string
isActive:    bool
createdAt:   datetime
updatedAt:   datetime
```

## Install

```
yarn
```

## Server

### Stack
* Webpack
* Koa
* Mongoose
* Jest
* Supertest

```
yarn server:start
```

Access localhost:4040.

#### Seed data into database
```
yarn seed
```

## Web
Buiding a table to praticing Material UI.

### Stack

* React/Hooks
* Typescript
* React Router
* Material UI

### Run

```bash
yarn web:start
```

In your browser access localhost:3030
